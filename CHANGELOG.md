# Changelog

## 0.0.2

- Added support for Template Fragment Caching `{% cache %}`.
- Added `as-cache` snippet for HTML templates.
- Added `as-cache-page` and `as-orm-cache` snippets for Python files.
- Enhanced syntax highlighting for template cache tags.
- **Improved Tailwind CSS support**: fixed IntelliSense in `class` attributes and added experimental support for `asok-class` directives.

## 0.0.1

- Initial release of Asok VS Code extension.
- Added snippets for Models, Routes, and Components.
- Added syntax highlighting for `asok-*`, `data-*`, and `ws-*` attributes.
- Integrated CLI commands in the Command Palette.
- Added HTML Data Provider for intelligent attribute autocompletion.
